# watson-server-starter
Server-side app for IBM Watson.

---

ES2017's async/await feature is also enabled.

|Compiler|version|
|:--|:--|
|typescript|2.1.4|

||My Environments|
|:--|:--|
|OS|Windows 10 Pro 64bit or 7 Pro 32bit|
|Editor|VSCode (stable)|
|node --version|v6.7.0|

## Setup
```
$ npm install
```

## Run (ts files directly)
```
$ npm start
```

## Build JS files
```
$ npm run build
```
